# HydraMate Smart Hydration System

A smart hydration tracking system that combines IoT hardware with a mobile application to encourage healthy hydration habits.

## Features

• Real-time hydration tracking

• Live synchronization using Firebase

• Smart bottle integration using ESP32

• Leaderboards

• Notifications

• Friends

• Daily goals

## Built With

React Native

Firebase Authentication

Firebase Realtime Database

ESP32

HX711
